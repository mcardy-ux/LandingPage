# Mcardi - Vuejs Landing Page
Landing Page personal donde muestro mis proyectos y articulos principales de mi blog.

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```
